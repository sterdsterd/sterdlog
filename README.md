# 소개

React + Typescript를 공부하고, 활용해보고 싶어 개인 웹사이트 겸 개발 블로그를 Gatsby로 만들었습니다.
여러 개발 블로그들을 참고하여, 개인적으로 블로그에 담고 싶은 기능들을 모두 담아보고자 했습니다.

# 기능

- LaTeX 지원
- 포스팅 검색 지원
- 다크 모드 지원
- 코드 하이라이팅 지원
- ToC 사이드바 지원
- giscus 댓글 지원
- Tossface 이모지 지원
- 태그 별 포스트 보기 지원
- mdx 지원

## 추후 개발할 기능

- 조회수 추가
- 깃허브, 연락처(이메일 등), 소셜 바로가기 추가
